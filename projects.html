Reseach: Automated Algorithm selection
In the MALLET lab I assist with a project that looks to improve automated algorithm selection. Traditionally, automated algorithm selection models are trained on the problem instance feature values and performance data from algorithm runs. The project I work on showed training these automated algorithm selection models on the feature values of the algorithms in addition to problem instance features improves overall performance. This allows the evaluation of sets of problem instances in less time and with less memory by choosing the most optimal algorithm more frequently. Extended abstract: Abstract.pdf
